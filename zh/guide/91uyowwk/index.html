案例8. 快速数据质量评估

1830 字约 6 分钟

2025-07-19

本案例展示如何使用DataFlow中GeneralText部分的算子进行快速质量评估。

1. 安装DataFlow环境

请通过如下命令安装DataFlow所需环境依赖。

pip install open-dataflow

2. 创建DataFlow工作区

用户可以在本地创建一个用于DataFlow快速运行的文件夹作为工作区。下面的指令提供了一个例子：

mkdir run_dataflow
cd run_dataflow

3. 初始化DataFlow

创建并切换当前文件夹到工作区后，用户可以一键初始化DataFlow的quick start运行代码

dataflow init

这时你会看到文件夹中出现了一系列测试代码及文件。

run_dataflow/playground/quick_evaluate.py

4. 配置LLM Serving

本案例所需算子需要基于API的LLM Serving，所以在运行之前需要把API key写入到环境变量中。 对于Linux和Mac OS系统

export DF_API_KEY="sk-xxxxx"

对于Windows系统

$env:DF_API_KEY = "sk-xxxxxx"

后续在代码中会通过环境变量读取该 API Key，无需在脚本中明文填写。

另外，api_url也需要在run_dataflow/playground/quick_and_evaluate.py中以如下方式填写：

self.llm_serving = APILLMServing_request(
        api_url="https://api.openai.com/v1/chat/completions",
        model_name="gpt-4o",
        max_workers=100
)

5. 一键运行

python quick_evaluate.py

运行后，该流水线会调用MetaScorer，进行基于预指定维度的多维数据评估。

关键参数说明

  • MetaScorer.dimensions需要评估的维度列表，其中每种维度对应一个字典，关键字有
    • dimension_name: 维度名称
    • description: 维度的具体描述
    • example_list: 示例列表，其中每个示例都遵循类似于{"text": "some text", "score": 1}的字典格式。
